It's been a while I was looking for some better solution to setup a blogging website and here I found something interesting. In this blog you are going to build free blogging website.

Github

Go Github.com and sign up for Github account. If you have already account then well and good.

Github Pages

For this you are going to use Github Pages service and it will work as a free website hosting service. On Github.com you need to setup your own free Github pages repository, e.g. https://github.com/YOUR_USERNAME/YOUR_USERNAME.github.io

Create your static website

In this blog we are going to setup pretty simple static website but you can setup fancy web site as well with GitHub Pages supported themes but here I am going to keep it simple. just creating Hello world HTML site.
Go to your local machine terminal and create following files and directories

$ mkdir <YOUR_GITHUB_USER_NAME>.github.io
$ cd <YOUR_GITHUB_USER_NAME>.github.io
$ touch index.html.

Then add following supper easy HTML code in your index.html file.

<html>
<header><title>Welcome to Github Pages.</title></header>
<body>
Hello world
</body>
</html>

Now in same directroy run following few command to create a local git repo and commit your code

$ git init
$ git add index.html
$ git commit -m "first commit"
$ git remote add origin git@github.com:<YOUR_GITHUB_USERNAME>/<YOUR_GITHUb_REPO_NAME>.github.io.git
$ git push -u origin master

Now you should be able to access webite with github.io URL. e.g. https://<YOUR_GITHUB_REPO_NAME>/.github.io that is https://example.github.io.
Some times it takes few minutes to deploy your changes. You might need to wait for 5-10 minutes here.

Custom domain for your GitHub Pages

If you are looking for custom domain for your GitHub Pages website then add CNAME file in your github repository as following and add your domain name in that CNAME file. e.g. example.com

$ touch CNAME
$ git add CNAME
$ git commit -m "Add CNAME"
$ git push -u origin master

Google Doamins

Purchase a domain at Google Domains at just $12 a year.

Once you purchase a domain then point it your github pages website. To do that go to "Configure DNS" tab on https://domains.google.com and scroll down to the "Custom resource records" section. You need to add A record which points to Github pages IP addresses.

Site is live now at https://example.com. Check it out.
